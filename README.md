# Ramadan Warmth

A small Ramadan-themed Next.js app with a calm illustration UI and a guided niyet flow.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Routes
- `/` home scene
- `/dua` guided niyet flow
- `/about` gentle note

## Local setup
```bash
npm install
npm run dev
```
Open `http://localhost:3000`.

## Build check
```bash
npm run build
npm run start
```
